### [Blocker] SQLite FTS5 External Content Table (`content='memory'`)의 데이터 동기화 누락
* **근거**: 
  - [plans/v0.2-memory-enhancement-plan.md:L44](file:///Users/drlee/workspace/pocket-llama-lab/plans/v0.2-memory-enhancement-plan.md#L44) (§1 데이터 모델: `CREATE VIRTUAL TABLE memory_fts USING fts5(text, content='memory')`)
* **이유**: FTS5에서 `content='external_table'` 방식을 사용할 때, SQLite는 원본 테이블(`memory`)의 변경사항(INSERT, UPDATE, DELETE)을 FTS 가상 테이블(`memory_fts`)에 자동으로 동기화해 주지 않는다. 트리거를 정의하거나 데이터 변경 시마다 동기화 쿼리를 매번 수동으로 실행하지 않으면 검색 인덱스가 갱신되지 않아 키워드 검색 폴백이 완전히 망가진다.
* **대안**: `memory` 테이블에 데이터가 들어오고 나갈 때 `memory_fts`를 실시간 갱신하는 SQLite TRIGGER 3종(INSERT/UPDATE/DELETE)을 설계서 DDL에 명시적으로 추가하라.
  ```sql
  CREATE TRIGGER tbl_ai AFTER INSERT ON memory BEGIN
    INSERT INTO memory_fts(rowid, text) VALUES (new.rowid, new.text);
  END;
  ```

## 3. Major

### [Major] UUID (TEXT PRIMARY KEY) 기본 키에 따른 FTS5 External Content rowid 매핑 한계
* **근거**: 
  - [plans/v0.2-memory-enhancement-plan.md:L30-L44](file:///Users/drlee/workspace/pocket-llama-lab/plans/v0.2-memory-enhancement-plan.md#L30-L44) (§1 데이터 모델: `id TEXT PRIMARY KEY`와 `content='memory'`)
* **이유**: SQLite FTS5 외부 콘텐츠 테이블은 참조 대상 테이블의 `rowid`(64비트 정수)와 1:1로 매핑되어 인덱싱을 수행한다. `memory` 테이블의 기본 키가 UUID(`TEXT`)로 정의되어 있으면 FTS5 가상 테이블의 내부 `rowid`와 대응되지 않아, 특정 데이터를 DELETE 하거나 UPDATE 할 때 FTS 테이블의 인덱스 일관성을 유지하기가 극도로 난해해진다.
* **대안**: `memory` 테이블에 `rowid INTEGER PRIMARY KEY AUTOINCREMENT`를 명시적으로 추가하고, UUID인 `id` 필드는 보조 `UNIQUE` 인덱스로 처리하라. FTS5 가상 테이블은 이 정수형 `rowid`를 기준으로 매핑 및 트리거를 정의해야 데이터 정합성 유지가 가능하다.

### [Major] FTS5 unicode61 토크나이저의 한국어 조사/형태소 매칭 한계
* **근거**: 
  - [plans/v0.2-memory-enhancement-plan.md:L15](file:///Users/drlee/workspace/pocket-llama-lab/plans/v0.2-memory-enhancement-plan.md#L15) (§0 `M-D4` SQLite FTS5 폴백)
* **이유**: SQLite 내장 `unicode61` 토크나이저는 문장 부호와 공백을 기준으로 단어를 분리한다. 한국어는 조사("~를", "~가", "~는")가 어근 뒤에 결합하는 교착어이므로, 사용자가 "매운 음식을 못 먹는다"라고 저장했을 때 "음식"으로 검색하면 매칭에 실패하여 폴백 검색의 실효성이 크게 떨어진다.
* **대안**: 키워드 폴백 가상 테이블 생성 시 `tokenize='trigram'` 옵션을 도입하여 한국어 조사 분리가 되지 않더라도 substring 부분 일치 검색이 가능하도록 사양을 정정하라.
